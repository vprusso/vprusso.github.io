# vprusso.github.io

Personal website built with [Hugo](https://gohugo.io/) using the [terminal](https://github.com/panr/hugo-theme-terminal) theme.

## Local Development

To run the site locally:

```bash
hugo server
```

This starts a local development server at `http://localhost:1313/` with live reload.
